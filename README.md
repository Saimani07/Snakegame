# 🐍 Retro Snake Game (Java Swing)

A classic **Snake Game** built using Java Swing with a modern retro design. This project features smooth gameplay, responsive controls, and a clean UI — all in a single Java file.

---

## 🚀 Features

* 🎮 Classic Snake gameplay
* 🖥️ GUI-based (Java Swing)
* 🎨 Retro dark theme with smooth graphics
* 🍎 Random apple generation
* 📈 Score tracking system
* ⚡ Increasing difficulty (speed increases over time)
* ⏸️ Pause / Resume functionality
* 🔄 Restart option
* ❌ Game Over screen with final score

---

## 🧠 How It Works

* The snake moves on a grid using a **timer-based loop**
* Apples spawn randomly on free positions
* When the snake eats an apple:

  * 🍎 Score increases
  * 🐍 Snake grows
  * ⚡ Speed increases gradually
* Game ends when:

  * Snake hits itself
  * Snake hits the wall
